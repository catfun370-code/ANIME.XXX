# ANIME LEGACY — أعظم عشرة أنميات في التاريخ

مجموعة من صفحات الويب الفاخرة تعرض أفضل عشرة أنميات في تاريخ الرسوم المتحركة اليابانية.  
كل صفحة بتصميم مختلف تماماً، مصممة بأعلى مستوى من الإبداع والفخامة.

---

## الملفات

### 1. `anime-immortals-final.html` ⭐ الأفضل
> **Split-Screen Editorial** — التصميم الأحدث والأكثر فخامة

- تصميم Split-Screen: Panel معلومات على اليسار + صورة سينمائية على اليمين
- **Aurora حية** تتغير بلون كل أنمي تلقائياً (ذهبي، أحمر، أزرق، بنفسجي...)
- **Typewriter Effect** — اسم الأنمي يتكتب حرف بحرف
- Loader بـ Clip-Path Wipe سينمائي
- Cursor مخصص بالكامل
- تنقل بـ Scroll / Swipe / Keyboard / Dots

## كيفية الاستخدام

الملفات كلها **HTML خالص** — لا تحتاج أي تثبيت أو سيرفر.  
فقط افتح أي ملف مباشرة في المتصفح.

```bash
# أو شغّله مباشرة
open anime-immortals-final.html
```

### GitHub Pages

لعرض الموقع أونلاين مجاناً عبر GitHub Pages:

1. ارفع المجلد على GitHub
2. اذهب إلى **Settings → Pages**
3. اختر **Branch: main** ثم اضغط **Save**
4. سيكون الموقع متاحاً على:  
   `https://USERNAME.github.io/REPO-NAME/anime-immortals-final.html`
